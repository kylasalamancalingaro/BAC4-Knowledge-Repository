# BA Agent (Local + Ollama)

This BA agent is intentionally stored outside your Global-Clients.wiki repo so it is not synced to Git.

## Location
C:\Users\kyla.salamanca\Documents\BA-Agent-Local

## What it does
- Summarize meeting notes
- Extract key decisions
- Extract business requirements
- Draft BRD updates from notes
- Mask common PII from transcripts/documents before sharing

## Prerequisites
1. Install Ollama: https://ollama.com/download
2. Pull model: ollama pull llama3.1
3. Keep Ollama running on http://localhost:11434

## Quick sample run
python "C:\Users\kyla.salamanca\Documents\BA-Agent-Local\ba_agent.py" summarize --input "c:\Users\kyla.salamanca\OneDrive - Lingaro Sp. z o. o\Compass\GCP\Global-Clients.wiki\Discovery\accounts\pg\pg-csat\06-meeting-and-email-notes\2026-04-11-pg-csat-dwmys-email-exchanges-apr-8-11.md" --output "C:\Users\kyla.salamanca\Documents\BA-Agent-Local\outputs\summary.md"

## PII masking script
File: `C:\Users\kyla.salamanca\Documents\BA-Agent-Local\pii_masker.py`

Masks common PII patterns in `.txt` / `.md` content:
- Email addresses
- Phone numbers
- SSN format (`123-45-6789`)
- Credit card-like numbers
- IPv4 addresses

### Usage
Write to a new file:

`python "C:\Users\kyla.salamanca\Documents\BA-Agent-Local\pii_masker.py" --input "<input-file>" --output "<masked-output-file>" --report`

Overwrite original file:

`python "C:\Users\kyla.salamanca\Documents\BA-Agent-Local\pii_masker.py" --input "<input-file>" --in-place --report`

Print masked content to console:

`python "C:\Users\kyla.salamanca\Documents\BA-Agent-Local\pii_masker.py" --input "<input-file>"`
